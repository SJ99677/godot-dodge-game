# 🎮 Dodge Game (Godot 2D)

A simple 2D game prototype built using Godot to demonstrate core game development concepts such as player movement, enemy AI, and collision detection.

## 🚀 Features

* Player movement using keyboard input (WASD / Arrow Keys)
* Enemy AI that tracks the player using vector-based movement
* Collision detection with automatic game reset
* Lightweight and easy-to-understand structure

## 🛠️ Built With

* Godot Engine (2D)
* GDScript

## 🎯 Purpose

This project was created to explore fundamental game mechanics and real-time interaction systems. It focuses on:

* Handling user input
* Implementing basic AI behavior
* Understanding game loops and physics updates

## ▶️ How to Run

1. Download or clone this repository
2. Open the project in Godot (version 4.x recommended)
3. Set up a simple scene with:

   * Player (CharacterBody2D)
   * Enemy (CharacterBody2D)
4. Attach the provided scripts
5. Run the main scene

## 📁 Project Structure

* `player.gd` – Handles player movement and collision logic
* `enemy.gd` – Controls enemy tracking behavior
* `project.godot` – Project configuration
